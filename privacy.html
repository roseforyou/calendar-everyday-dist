隐私权政策 (Privacy Policy)

生效日期：2025年12月30日

“天天日历扩展”（以下简称“本插件”）尊重并保护所有使用服务用户的个人隐私。为了给您提供更准确、更优质的服务，本插件会按照本隐私权政策的规定使用和披露您的相关信息。

1. 我们收集的信息

本插件属于纯工具类产品。我们在您使用插件的过程中：

  • 不收集您的任何个人身份信息（如姓名、电话、电子邮件等）。
  • 不收集您的浏览记录、搜索历史或任何页面内容数据。
  • 不收集任何地理位置、账号信息或联系人数据。

2. 我们如何使用信息

本插件的主要功能是提供公历、农历和节假日的查看及换算。

  • 远程数据访问：本插件仅访问特定的公共 API (主要来自 shuyz.com 和 timor.tech) 以获取最新的法定节假日安排和农历信息。这些请求是匿名发起的，不包含任何个人标识符、Cookie 或跟踪信息。
  • 本地存储：您的任何个性化设置（如自定义主题或标记）均仅通过 Web Storage API 存储在您的本地浏览器中。这些数据不会同步到我们的服务器，也不会发送给任何第三方。

3. 数据安全与第三方共享

  • 我们绝不会将您的数据出售给第三方。
  • 本插件不包含任何广告追踪、分析统计工具或第三方 SDK。
  • 所有的代码逻辑均已打包在扩展程序内，不包含任何远程执行的脚本。

4. 政策修订

我们可能会适时修改本隐私政策。如果本政策发生重大变更，我们会在 GitHub 仓库中更新。如果您继续使用本插件，即表示您同意受经修订的隐私政策的约束。

5. 联系我们

如果您对本隐私政策有任何疑问、意见或建议，请通过本项目的 GitHub 仓库提交 Issue 或联系开发者。
